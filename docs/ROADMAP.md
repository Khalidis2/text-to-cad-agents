# Roadmap

## Phase 1: Foundation

- Lock product specification.
- Keep the architecture local-first.
- Preserve the two-agent workflow.
- Generate one product family reliably.
- Store every job with a full report.
- Keep CAD generation deterministic.

## Phase 2: Better first product family

- Improve car mirror hanger proportions.
- Improve typography handling.
- Add Arabic text path/vector workflow.
- Add more preview options.
- Add stronger geometry checks.
- Add slicer-oriented print recommendations.

## Phase 3: Product template expansion

Add product families in this order:

1. Keychain
2. Nameplate
3. Cable clip
4. Organizer tray
5. Simple box/container
6. Phone stand
7. Wall hook
8. Filament label/clip

Each template must include defaults, constraints, validators, examples, and tests.

## Phase 4: User interface

- Local web interface.
- Job history page.
- Preview and download page.
- Telegram bot after local engine is stable.

## Phase 5: Intelligence adapters

- Deterministic local fallback.
- Local LLM adapter.
- Optional OpenAI adapter.
- Shared structured schema so the CAD engine is independent from the AI provider.

## Phase 6: Advanced production features

- Auto-fix loop.
- Visual critic using rendered previews.
- Cost estimator.
- Material estimator.
- BOM generator.
- Assembly support.
- Reference image input.

## Long-term vision

The project should grow into a local-first CAD production system that can turn specified product ideas into printable, editable CAD files with strict validation.
